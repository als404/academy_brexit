Работаем с программой GoLabel (Печатаем наклейки на товар)

ШАГ 1 — запуск программы

Запускаем программу! Для запуска программы GoLabel вам надо найти на рабочем столе ярлык программы и кликнуть по нему два раза левой кнопкой мышки


ШАГ 2 — подготовка к работе

После загрузки программы надо закрыть окно "Настройка страницы" (т.к. у нас уже настроены размеры наклейки).


ШАГ 3 — открываем готовый шаблон

У нас уже есть готовые шаблоны наклеек. Чтобы открыть один из шаблонов, кликаем по кнопке "Открыть"  и выбираем шаблон из списка!

Шаблон выбираем, исходя из того, на какой бренд нам надо напечатать наклейку!


Переходим на уровень вверх и выбираем директорию с нужным брендом, например, выберем бренд Virax!
Заходим в поддиректорию и видим список с уже готовыми шаблонами!


Названия шаблонов — это артикул товара либо его название!

ШАГ 4 — копирование шаблона

Если нужного для вас шаблона нет, делаем копию любого шаблона, для этого надо нажать правой кнопкой мышки и выбрать пункт Копировать


После этого опять нажимаем правую кнопку мышки и выбираем пункт Вставить скопированный шаблон в эту же директорию


После этого нам надо переименовать этот шаблон. Дадим ему новое имя, например артикул нужного нам товара.

Для этого кликаем правой кнопкой мышки по новому шаблону и выбираем пункт Переименовать


ШАГ 5 — верстка шаблона

Открываем созданный вами новый шаблон и редактируем его


Изменяем название товара.


Кликаем два раза по названию, в открывшимся окне находим поле с названием "Текстовые данные", в этом поле мы можем изменить название товара, удаляем содержимое этого поля и пишем в него наше название. Нажимаем ОК.


Далее изменяем артикул и модель товара (если она присутствует). Все действия такие же, как при изменении названия.

Меняем штрих-код. Для этого кликаем два раза левой кнопкой мылки по штрих-коду


в открывшемся окне находи поле с данными штрих-кода


Выписываем в поле номер штрих-код для нашего товара. Нажимаем кнопку ОК.

Изменяем QR-код. Для этого кликаем два раза, левой кнопкой мышки, по QR-коду, откроется окно в котором мы ищем поле "Bar Code Data"


удаляем содержимое этого поля и вставляем туда ссылку на наш товар!

Ссылку на товар находим на нашем сайте discount-tools.ru: для этого открываем браузер и заходим на сайт компании


Сверху находим поле Поиск и в пишем туда артикул нужного нам товара


В сплывающем окне находим наш товар и один раз кликаем, левой кнопкой мышки, по нему, после этого откроется окно с выбранным вами товаром


Далее копируем из строки адреса нашу ссылку на товар. Для этого кликаем один раз, правой кнопкой мышки, по строке с адресом страницы и выбираем пункт Копировать


после этого вставляем нашу ссылку в поле "Bar Code Data" и нажимаем кнопку ОК


ШАГ 6 — сохраняем шаблон

Сохраним наш шаблон. Для этого нажмите кнопку Cохранить 

ШАГ 7 — печать наклейки

Печать наклейки. Для того чтобы напечатать нужное нам кол-во наклеек, нам надо открыть "Настройки печати" 

Откроется окно с настройками печати


находим поле "Кол-во для печати" пишем туда нужное кол-во штук


после этого нажимаем кнопку Печать 

После этого можно закрывать данное окно, нажав на кнопку Выход 

ШАГ 8 — закройте программу

После того, как работы с программой завершены выполните выход из программы (закройте программу)!
