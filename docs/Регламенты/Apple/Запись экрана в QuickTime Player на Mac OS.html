Как можно делать видеозапись того, что отображается на экране компьютера?

ШАГ 1 — запуск программы

В программе QuickTime Player на MacOS выберите меню "Файл" > "Новая запись экрана".

для MacOS до версии 10.14 (Mojave)


для MacOS от версии 10.15 (Catalina)


Выберите "Запись выбранной области" и задайте размер записываемого окна

ШАГ 2 — настройка программы

для MacOS до версии 10.14 (Mojave)

Нажмите всплывающее меню "Параметры", а затем выберите любой из следующих параметров записи:

  • Микрофон. Выберите "Нет", чтобы записать отображаемое на экране изображение без звука, либо выберите микрофон.
  • Параметры. Выберите "Показывать нажатия мышью в записи", чтобы все нажатия, которые производятся во время записи изображения на экране, отображались на видеозаписи в виде кружочка.


для MacOS от версии 10.15 (Catalina)

Нажмите всплывающее меню "Параметры", а затем выберите любой из следующих параметров записи:

  • Микрофон. Выберите "Нет", чтобы записать отображаемое на экране изображение без звука, либо выберите микрофон.
  • Параметры. Выберите "Показывать нажатия мышью в записи", чтобы все нажатия, которые производятся во время записи изображения на экране, отображались на видеозаписи в виде кружочка.
  • Место сохранения. Выберите директорию куда сохранится файл после остановки записи.


ШАГ 3 — запись

для MacOS до версии 10.14 (Mojave)

Нажмите кнопку "Запись" после чего выберите тип записи.

  • Нажмите один раз в любом месте экрана, чтобы начать запись всего экрана.
  • Потяните, чтобы выделить часть экрана, которую требуется записать, а затем нажмите кнопку "Начать запись". Например, можно обвести курсором мыши только определенное окно и записать его.

для MacOS от версии 10.15 (Catalina)

Нажмите кнопку "Запись"

ШАГ 4 — стоп

Закончив запись, нажмите кнопку "Стоп" в строке меню (или используйте Touch Bar).


ШАГ 5 — сохранение

для MacOS до версии 10.14 (Mojave):

Выберите "Файл" > "Сохранить", затем введите имя записи.

для MacOS от версии 10.15 (Catalina):

Файл сохранится в выбранную директорию выбранную в параметрах.


Запись сохраняется в формате фильма QuickTime H.264 с звуковой дорожкой AAC (если звук записывался).
