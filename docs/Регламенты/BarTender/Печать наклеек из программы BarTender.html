ШАГ 1 – запускаем программу BarTender

Для этого кликните левой кнопкой мышки по ярлыку программы на рабочем столе.

ШАГ 2 – открываем макет наклейки

Выбираем Файл – Открыть (CTRL + O) 


В открывшемся окне выбираем нужный нам шаблон шаблон например 4000120.btw и нажимаем кнопку "Открыть".


Либо создаем новый редактируя уже готовый макет. Для этого лучше сделать копию уже готового макета который вам подходит.

Далее нажимаем Файл – Печать, в открывшемся окне на вкладке "Печать" проверяем что бы в поле "Имя" был выбран принтер "Xprinter XP-420B", вкладка "Количество", в поле "Копии": ставим кол-во наклеек и нажимаем кнопку "Печать".


Началась печать наклеек на принтере.
