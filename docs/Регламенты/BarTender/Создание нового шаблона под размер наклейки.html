ШАГ 1 — запуск программы

Для этого кликните левой кнопкой мышки по ярлыку программы на рабочем столе.

ШАГ 2 — создаем новый документ

После загрузки программы в открывшемся окне выбираем пункт "Создать новый документ BarTender..." 


Выбираем "Пустой шаблон" и нажимаем кнопку "Далее"


Затем выбираем пункт "Установить специальные параметры" нажимаем кнопку "Далее"


Посмотрите на бабину с наклейками, если на ленте всего один ряд наклеек выбираем пункт "Один элемент на странице" и нажимаем кнопку "Далее".


Если между краем подложки и краем наклейки есть растояние выбираем пункт "Да, по краям должены оставаться неиспользуемые области", и замеряем это расстояние от края подложки до наклейки с левой и правой стороны. 


Заполняем поля: в поле "Левое" = 3,0 мм и в поле "Правое" = 2,0 мм и нажимаем кнопку "Далее".


Далее выбираем формат печатаемого элемента, например наклейка у нас прямоугольная и у нее скругленные углы, тогда выбираем пункт "Скругленный прямоугольник" и нажимаем кнопку "Далее".


Далее замеряем размер самой наклейки (у нас наклейка 75х120 мм), и вписываем в поле "Ширина" = 75,0 мм, и поле "Высота" = 120,0 мм. Если информация будем печатать по длиной стороне наклейки, то выбираем пункт "Альбомная" и нажимаем кнопку "Готово"


После этого откроется окно с шаблоном наклейки с параметрами которые были задали выше.


ШАГ 3 — сохранение документ

Что бы в последующем не создавать шаблон заново (с такиме же параметрами) сохраните пустой шаблон на диск (D:/_barcode_/), для этого выберете пункт Файл – Сохранить(либо нажмите сочетание клавиш CTRL + S)


В открывшемся окне задайте новое имя для шаблона, например его размер: 75x120.btw и нажмите кнопку "Сохранить".
