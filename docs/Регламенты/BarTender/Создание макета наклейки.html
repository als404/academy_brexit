Изучение данного регламента поможет понять, как создавать и редактировать макет наклейки! 

Запускаем программу BarTender UltraLite

Для этого кликните левой кнопкой мышки по ярлыку программы на рабочем столе.

ШАГ 1 – открываем пустой шаблон

Открываем пустой шаблон наклейки с размерами под вашу наклейку, который вы создали зарание по регламенту" Создание нового шаблона под размер наклейки". Для этого выбираем Файл – Открыть (CTRL + O) 


В открывшемся окне выбираем пустой шаблон 75x120.btw и нажимаем кнопку "Открыть".

Открылся шаблон документа на котором мы будем создавать макет будующей наклейки.


ШАГ 2 – добавляем заголовок

Теперь добавлим название товара, для этого нажимаем на пункт Создать – Текст и выбираем пункт "Одна строка" и кликаем левой кнопкой мышки по рабочей области наклейки, после чего там появится поле с текстом в одну строку.


Далее кликаем по этому полю левой кнопкой мышки, удаляем текущий текст, и набираем на клавиатуре, либо вставляем туда заранее скопированное в буфер обмена,  название товара. Например "Передавливатель для труб механический".


Параметры текста: шрифт "Myriad Pro Cond", размер шрифта 14, стиль начертания жирный 


ШАГ 3 – добавление названия модели

Для вставки названия модели товара повторяем ШАГ 2. 

Параметры для текста: шрифт "Myriad Pro Cond", размер шрифта 22, стиль начертания жирный  .


ШАГ 4 – добавление артикула

Добавляем артикул товара для этого повторяем ШАГ 2. 

Параметры для текста: шрифт "Myriad Pro", размер шрифта 26, стиль начертания жирный, выбираем пункт "Белый на черном" (CTRL + W) текст центрируем по центру.


ШАГ 5 – добавляем штрих-код

Что бы добавить штрих-код, кликаем левой кнопкой мышки по пункту Создать – Штрих-код и в появившемся окне выбираем в выпадающем списке "Недавно использованные штрих-коды" EAN-13, если в выпадающем списке нету такого названия, нажимаем на пункт "Дополнительные штрих-коды" и в открывшемся окне в поле поиска набираем название EAN-13.


После этого будет найдено несколько вариантов, выбираем в списке EAN-13 и нажимаем кнопку "Выберите"


Далее кликаем левой кнопкой мышки по рабочей области, что бы добавить штрих-код.


Далее нам надо от редактировать штрих-код. Двойным нажатием левой кнопкой мышки по штрих-коду открываем окно "Свойства штрих-кода" и приступаем к изменению свойств объекта. 


В открывшемся окне переходим в пункт "Источники данных" и кликаем по номеру штрих-кода 2112345678917


Штрих-код можно взять в прайс-листе компании Канюк на Google.Drive. Он находится в первой колонке А напротив товара нужного вам товара. 


Если в колонке нет штрих-кода, его нужно заказать: для этого создайте новую задачу на портале Битрикс24, где ответственный: Инна Васильева, текст описания задачи: "Нет штрихкода на товар: бренд "Брекзит", артикул: *******, закажите пожалуйста новый штрих-код и добавьте в прайс-лист".

В поле "Встроенные данные" вставляем из буфера обмена номер штрих-кода нашего товара 4631164005295. 


Далее добавляем границу обриса для штрих-кода, для этого надо перейти в пункт "Граница" и добавить значения как на картинке ниже: Тип границы: Прямоугольник; Поля: Верхнее = 2,0 мм; Левое = 3,0 мм; Нижнее = 2,0 мм; Правое = 3,5 мм; Свойство линии: Толщина = 0,3 пт.


Теперь зададим стили для цифр штрихкода. Для этого надо перейти в пункт Шрифт: выбираем шрифт = Myriad Pro, начертание = Обычный, Размер = 10


Теперь перейдем в пункт "Положение" и добавим данные для расположения штрих-кода в рабочей области документа. Выставляем в поля "X" = 10,5 мм; "Y" = 41,2 мм; "Контрольная точка" объекта "Сверху слева". 


Далее кликаем по кнопке "Закрыть".


ШАГ 6 – добавляем знак сертификата ЕАС

Для этого нажмите пункт Создать - Рисунок и выберите пункт вставить из файла, в открывшемся окне выберите файл и нажмите кнопку "Выбрать"


Затем надо переместить добавленную картинку на задний план. Для этого кликнете правой кнопкой мышки на картинку и выберите пункт Упорядочивание – Заказ – На задний план


ШАГ 7 – добавление логотипа

Для вставки логотипа компании Брекзит выполните ШАГ 6, только при выборе файла кликните по файлу с логотипом.


ШАГ 8 – добавляем адрес компании

Добавим текст: "Производитель" для этого выполните ШАГ 2


Параметры текста: шрифт "Myriad Pro Cond", размер шрифта 10, стиль начертания жирный 

Теперь добавим текст "Адрес компании", для этого выполните ШАГ 2,

 

Параметры текста: шрифт "Myriad Pro Cond", размер шрифта 10, стиль начертания обычный (кнопка начертания должна быть не активна)


ШАГ 9 – добавление QR-кода

Для этого кликаем по Создать – Штрих-код в появившемся окне выбираем в списке "Недавно использованные штрих-коды" QR-code, если в списке такого названия нет нажимаем на пункт "Дополнительные штрих-коды".


В открывшемся окне в поле поиска вписываем QR-code

После этого выбираем в списке название QR-code и нажимаем кнопку "Выберите"


Далее кликаем левой кнопкой мышки по рабочей области, что бы добавить QR-код.


Теперь надо изменить свойства для нашего QR-кода для этого кликните два раза левой кнопкой мышки на QR-код


В пункте "Читаемость", пункт "Видимость" устанавливаем в значение "Нет"


Задаем границы: значение для всех полей = 3,0 мм, "Свойство линии" Толщина = 0,3 пт.


Переходим в пункт Положение и меняем поля: Х = 88,0 мм; Y = 7,6 мм, "Контрольная точка объекта": "Сверху слева".


Далее переходим в пункт "Источник данных", кликаем на номер 12345678 и в поле "Встроенные данные" вставляем URL нашего товара


Где взять URL товар?

Перейдем в браузере на сайт компании БРЕКЗИТ www.prof-inst.by

В поле поиск вставляем наш артикул, например: 4000080 и в выпадающем списке выбираем наш товар. т.е. кликаем левой кнопкой мышке по товару в выпадающем списке.


После загрузки страницы с товаром надо скопировать URL страницы, в буфер обмена, из поля адреса вашего браузера, для этого выделяем весь адрес, кликаем правой кнопкой мышки и в выпадающем меню выбираем пункт "Копировать" (CTRL + C)


Далее переходим назад в нашу программу и вставляем в поле "Встроенные данные" URL страницы с товаром, т.е. скопированые данные.


Далее перейдите в пункт "Символика и размер" в поле "Х-размерность" выбираем значение = 1,13 мм, (это подходит для QR-кода в котором короткий URL, например только адрес сайта)


Если после этого QR-код на шаблоне увеличился и вышел за границы страницы в пункте "Символика и размер" параметр для "Х-размерности" уменьшаем до того пока он не уменьшится QR-код до оптимальных, например до 0,50 мм и нажимаем кнопку "Закрыть"


Шаблон с наклейкой готов.

Сохраняем шаблон на диск, пункт Файл – Сохранить как переходим в папку с размером шаблона 75х120, если такой папки нету, создаем её, далее в поле "Имя файла" в место названия шаблона вписываем артикул "4000080.btw" и нажимаем кнопку "Сохранить".


На этом создание макета наклейки готов.
