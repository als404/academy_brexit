Установка и настройка приложение Groundwire для Android

ШАГ 1

Качаете и устанавливаете приложение Groundwire

ШАГ 2

Запускаете и нажимаете на три вертикальные точки


ШАГ 3

Выбираете "Учетные записи"


ШАГ 4

Нажимаете на "Новый SIP-аккаунт"


ШАГ 5

Вводите данные:

Название: ваш логин
Имя пользователя: ваш логин
Пароль: ваш пароль
Домен: pbx.zadarma.com

После заполнения всех данных нажимаете на значок галочки (вверху справа)


ШАГ 6

Переходите на главный экран (экран с клавиатурой). Если ввели все данные верно, то вверху слева должен быть зеленый индикатор, который говорит об успешной авторизации, и ваше отображаемое имя.


ШАГ 7

Снова заходите в настройки через три вертикальные точки, далее в разделе "Входящие звонки" нажимаете на пункт "Входящие звонки" и, вместо "Push-уведомления", выбираете "Стандартные".

Приложение необходимо держать открытым, иначе входящие звонки не будут поступать. Для того, чтобы проверить, работает ли приложение, вам нужно следить за зеленым индикатором в строке состояния. Если он пропал, значит приложение закрыто.

ШАГ 8

Совершаете тестовый звонок на 4444 для оценки качества голоса. В начале вам нужно оценить качество голоса робота, после, когда робот замолчит, вам нужно сказать пару слов для оценки вашего голоса.

Для перевода звонка

В текущем вызове набрать

#внутренний номер абонента# - для обычного перевода звонка без предварительного соединения с абонентом

*внутренний номер абонента# - для соединения с абонентом для диалога (при этом входящий звонок ставится на удержание) после чего ненужный абонент кладет трубку
