  • Зарегистрировать аккаунт Google;
  • Создать новый канал YouTube на тот же аккаунт;
  • Заполнить название канала (тематическое и уникальное с высокочастотным ключевиком, либо с несколькими);
  • Заполнить описание канала (начать с привлекательной фразы, по ключевикам, использовать ВЧ запросы, свой уникальный текст);
  • Загрузить привлекательный аватар канала [ВАЖНО (увеличивает конверсию)+;
  • Загрузить привлекательную шапку канала (в шапке расписать самые важные смыслы о компании, дату работы магазинов, чем этот канал будет полезен, визуально показать на блок ссылок с контактами) [ВАЖНО (увеличивает конверсию)+;
  • Ссылки в шапку (социальные сети, сайт и т.д.);
  • Записать приветственный трейлер канала;
  • Назвать и описать трейлер (оптимизировать ролик);
  • Добавить интересные каналы; (Вы можете показываться у кого-то в интересных каналах, только если сами включили эту функцию).
  • Добавить плейлисты и их описание (уникальное название и описание с ключами, использовать спец. символ «★» для усиление внимания, как минимум 10 видео); Один плейлист – один ключ. Под каждый товар или группу товаров можно создать свой плейлист, даже, если в нём будет только одно видео.
  • Настроить фид канала (шестерёнка на главной странице канала). Сверху добавить загрузки и популярное, далее плейлисты;
  • Подтвердить по СМС. Верифицировать канал (без этого не даст загружать превью).
  • Творческая студия –> настройки:
    • Заполняем ключевые слова для канала (используем, ВЧ запросы из описания);
    • Выключить рекламу на канале.
    • Отключаем возможность видеть кол-во подписчиков. Пока их не будет 300-400+ Можно аккуратно накрутить (piar4you.com).
    • Загрузка видео:
      • Поставить брендирующе теги ( название вашей компании в русской и английский раскладке, компании конкурентов, брендов продукции, имена по которым могут искать)
      • Доступ – Доступ по ссылке)
      • Комментарии – разрешены;
      • Написать шаблон описания для каждого видео (Оффер, преимущества, пара предложений о компании, контакты иссылки на соц сети, хештеги (те же теги, только через # и в описании)).
      • Выбрать категорию;
  • Создать кабинет Google рекламы;
  • Подключить Google рекламы к каналу;
  • Выключить монетизацию (не нужно для бизнес аккаунтов);
  • Добавить ссылки на соцсети в шапку канала;
  • Персональный URL канала (доступен после 100 подписчиков, 30 дней существования канала, на канале загружено фото профили и фоновое изображение для видео)

ОПТИМИЗАЦИЯ КАНАЛА:

  • правильное имя канала – наличие ключевых слов, никаких цифр в названии в информации о канале пропишите ключевые слова;
  • внимание на теги;
  • канал надо обязательно заполнить видеороликами и плейлистами;
  • Можно проводить конкурс на канале. С помощью одного такого розыгрыша можно получить лавину новых подписчиков.
  • Создаем плейлисты (под СЧ ключевики, они же рубрики канала): описание делать как можно длиннее;
  • повторить название плейлиста в первой строке.

ИДЕИ ДЛЯ ВИДЕО:

  • Главный принцип: ВИДЕО ДОЛЖНО ВЫЗЫВАТЬ ИНТЕРЕС. Задавать себе вопрос: кто его будет смотреть то? Зачем его смотреть?
  • Идеи можно брать из:
    • запросов под видео из комментариев зрителей.
    • поисковой строки Google и Яндекс (Через сервисы: для Google, для Yandex).
  • Запросы YouTube подсказках (в инкогнито, частотность – вордстат);
  • Профессиональные форумы
  • Видео в топе YouTube по нашей теме;
  • Анализ тематик других каналов;

ПЕРЕД НАЧАЛОМ ЗАГРУЗКИ:

  • качественный анализ ключевых слов;
  • качественное видео;
  • В текстовом документе заранее готовим:
    • название видео – ключевое слово (ключ, либо несколько ключей + желтуха (привлекающее), + повтор в другой словоформе , больше возможностей выйти в топ (через разделитель);
    • никогда не ставить первые цифры или символы только в конце;
    • в описании в 1й строке повторить название ролика;
    • описание видео (в 1й строке повторить название, в 2й строке — призыв к действию + ссылка; можно транскрибировать видео, 500-1000 символов о чем видео символов, ссылки на ресурсы 5-8, о чем канал 500-700 в первых 2х абзацы 2-3 ключевых слова, используем #хештеги);
    • добавить ссылки на другие видео (в описании);
    • прописать тайм-код (в описании);
    • активные обратные ссылки на социальные сети и сайты (в описании);
    • ключевые слова (ключевое слово — первый тег, брендовый – второй тег, и другие ключевые слова);
  • Готовим тексты и картинки в посты для социальных сетей, которые опубликуем сразу после выпуска ролика;
  • Готовим письмо для рассылки по подписной базе, о выходе нового видео.
  • Готовим сообщение для друзей;
  • Готовим красочное, привлекательное превью (картинка на видео) (переименовать аналогично названию видео файла, 1280*720);

ПРОДВИЖЕНИЕ ВИДЕО. ЧТО ВАЖНО:

  • привлекательная заставка (видео с примером создания);
  • Высокое удержание аудитории (более 50%) Больше динамики, меньше воды, больше полезной информации. Если ролик можно сделать короче, без вреда смыслу — сделать;
  • анализ конкурентов и подбор ключей (те ключи, которые пересекаются в конкурентов в топе видео по ключевому запросу, так же и с названием видео);
  • большое количество комментариев;
  • лайков больше чем дизлайков;
  • очень большая аутивность в первые часы после загрузки (просмотры, комментарии, лайки, репосты) за 2-24-48 часов

ВО ВРЕМЯ ЗАГРУЗКИ:

  • В названии писать некоторые слова ПРОПИСНЫМИ БУКВАМИ, для выделения из общей массы роликов. Использовать разделители между ключами\фразами «//», «,» , «[]», «|»;
  • Добавляем теги к видео (обязательно должны быть теги полностью повторять поисковые запросы, которыми мы хотим продвигаться);
  • Добавить ролик в плейлисты;
  • Создать опрос в видео согласно контексту ролика. Добавить конечную заставку. Вставить туда предложение подписаться на канал и 1-2 ролика с канала;
  • Добавить через отсроченную публикацию (отсроченная публикация не влияет на продвижение, когда видео опубликуется – начнётся отсчет);
  • Загрузить превью.
  • Если есть добавить текст субтитров;

ПОСЛЕ ЗАГРУЗКИ:

        «КЛЮЧЕВОЙ ЗАПРОС 1»
        «КЛЮЧЕВОЙ ЗАПРОС 2»
        «КЛЮЧЕВОЙ ЗАПРОС 3»

        С какого года работает консультант Андрей Викторович с Вольво?
        + Прислать скриншот вашего комментария».


  • Ставим автоподнятие раз в 1-5 минут первые пару часов.
