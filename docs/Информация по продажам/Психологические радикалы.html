Это особенности личности, при которых отдельные черты характера чрезмерно усилены. Важно: в человеке может сочетаться несколько радикалов и даже все, но в любом случае один-два из них доминирует.

Суть методики состоит в возможности оценки человека без проведения различных психологических тестов, для того, чтобы выявить особенности взаимодействия с ним.

Методика делит людей на 7 радикалов характера:

  1. Истероидный (демонстративный)
  2. Паранойяльный (целеустремленный);
  3. Эпилептоидный (бывает 2х видов: застревающий и возбудимый);
  4. Шизоидный (странный);
  5. Гипертимный (жизнерадостный);
  6. Эмотивный (чувствительный);
  7. Тревожный (боязливый).

Предлагаемая методика будет разбирать каждый радикал на следующие составляющие:

  • Общая характеристика
  • Внешнией вид
  • Качества поведения
  • Модель позитивного общения
  • Резюме
  • Мимика
  • Жесты
  • Речь
  • Идентификация клиента
  • Специфика коммуникации
