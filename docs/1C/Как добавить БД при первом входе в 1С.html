Шаг 1 — запуск программы

Запускаем программу 1С, для этого надо кликнуть два раза по значку программы 1С на рабочем столе! Откроется окно с выбором базы данных (БД)


Шаг 2 — выбор/создание базы данных

Добавляем нашу БД в список, для этого надо нажать на кнопку "добавить". В следующем окне выбираем пункт "Добавление в список существующей информационной базы" нажимаем кнопку Далее >


Шаг 3 — редактирование подключение к базе данных

В поле "Укажите наименование…" вписываем название базы, например 1С. Далее отмечаем пункт "На сервере 1С:Предприятие" и заполняем поля:


для компании Канюк:

  • Кластер серверов: srv-db
  • Имя информационной базы: base1c

для компании Брекзит:

  • Кластер серверов: srv-sql
  • Имя информационной базы: brexit

для компании ПрофИнст:

  • Кластер серверов: srv-sql
  • Имя информационной базы: onecdb1

и нажимаем кнопку Далее > в следующем окне нажимаем кнопку Готово

На этом настройка БД закончена, можно входить в 1С.
