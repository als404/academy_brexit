Шаг 1 —настройка подключения

Запускаем приложение Microsoft Remote Desktop. И добавляем новое подключение нажать на + и выбрать Add PC

В открывшемся окне заполняем поля:

для компании Канюк:

  • в поле PC Name: надо вписать IP-адрес 82.209.232.135:49033
  • в поле Friendly name: пишем название нашего подключения, например: Канюк 1С


для компании Брекзит:

  • в поле PC Name: надо вписать IP-адрес 82.209.232.135:49031
  • в поле Friendly name: пишем название нашего подключения, например: Брезит 1С


Шаг 2 — добавление пользователя

Добавляем нового пользователя, для этого надо в поле "User account" выбрать пункт "Add User Account…".

В открывшемся окне заполняем поля (данные которые вам предоставил системный администратор):

  • Username: ваш логин для доступа к серверу 1С
  • Password: ваш пароль
  • Friendly name: вписываем имя, которое будет отображаться при создании нового подключения, например User

После заполнения полей станет активна кнопка "Add", нажимаем эту кнопку.


Шаг 3 — сохранение настроек

После добавления пользователя сохраняем наше подлючение, для этого нажимаем кнопку Add , данные сохранились в Microsoft Remote Desktop.


На этом наше подключение настроено и готово к работе.

Дополнительно:

Шаг 4 — побрасываем принтер

Для настройки принтера нужно отредактировать удаленное подключение для этого выбираем в приложении Microsoft Remote Desktop наше подключение и нажимаем на карандаш либо выбираем из выпадающего списка пункт Edit.


В открывшемся оне переходим на вкладку Devices & Audio


далее отмечаем пункт Printers


если дальнейшие настройки не требуются нажимаем кнопку Save.

После следующего подключения к удаленному рабочему столу принтеры, которые подключены к вашему компьютеру автоматически пробросятся в систему и отобразятся в принтерах при печати из 1С (если нажать сочетание кнопок Ctrl+P можно увидеть список принтеров, где у принтеров подключеных у пользолвателя в скобках будет написано "перенаправлено")

нажав правую кнопку мыши в выпадающем меню можно выбрать свой принтер "по умолчанию"


Шаг 5 — подключаем удаленную папку

Если Вам потребуется сохранить файл с удаленного компьютера на свой компьютер для этого нужно отредактировать удаленное подключение, для этого выбираем в приложении Microsoft Remote Desktop наше подключение и нажимаем на карандаш либо выбираем из выпадающего списка пункт Edit.


Переходим на вкладку Folders и активируем чекбокс Redirect folders, снизу нажимаем на кнопку (+) и выбираем место где будут сохраняться файлы, нажимаем кнопку Open. После этого у Вас появится папка для хранения файлов. Нажимаем кнопку Add, а затем нажимаем кнопку Save.


Теперь при следующем подключении к удаленному рабочему столу на нем должен появиться диск в который Вы сможете сохранять нужные вам файлы.

Как сохранить документ из 1С на локальный компьютер

Далее при работе в 1С для сохранения документа на свой компьютер, нажмите в программе 1С кнопку "Сохранить на диске"


в открывшемся окне выбрать путь к месту куда вы хотите сохранить данный файл для этого нажмите на кнопку с 3-мя точками (либо клавишу F4 на клавиатуре)


в открывшемся окне в правой части выберите пункт "Этот компьютер" затем в основном окне в пункте "Перенаправленные диски и папки" выберите ваш диск (выберите папку куда хотите сохранить файл)


далее нажмите кнопку "Выбор папки", окно закроется и в строке появится полный путь к месту куда будет сохранятся документ. Далее нужно выбрать интересующий вас формат документа и нажать кнопку "Сохранить"


Файл должен быть сохранен на вашем компьютере в указаной вами папке!
